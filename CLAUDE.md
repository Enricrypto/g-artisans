## Architecture Rules

### 1. Design System â€” Non-Negotiable
- **Colors:** Use Tailwind tokens defined in `tailwind.config.ts` (Naranja, Selva, Linho, Noche, Muted, Rule)
- **Typography:** Cormorant Garamond (display, serif) + Jost (body, sans-serif) â€” no other fonts
- **Buttons:** 4 variants only (Primary Orange, Secondary Green, Outline Orange, Outline Dark) with `border-radius: 40px`
- **Badges:** Pill shape with `border-radius: 20px`
- **Icons:** Lucide React only â€” no emoji in production UI

Reference: Section 10 of TRD (DiseĂ±o â€” Tokens y tipografĂ­a)

### 2. Forms & Validation
- Producer application form (`/apply`) is THE critical component
- Use React Hook Form + Zod for validation (both client-side and server-side)
- Honeypot field `_gotcha` for anti-spam (positioned off-screen, not `display:none`)
- Rate limiting: Max 3 requests/hour per IP via `@upstash/ratelimit` or in-memory Map

Reference: Section 3 (Formulario de solicitud de productor)

### 3. APIs & Data Flow
- **Airtable:** Two tables: `Productores_Solicitudes` + `Compradores_Interesados`
- **Resend:** Two email types: producer confirmation + internal notification
- **Retry logic:** Exponential backoff for Airtable (max 3 attempts, handles 429)
- **Caching:** `/api/spots` uses ISR `revalidate: 60`

All API routes defined in Section 4 (EspecificaciĂ³n de APIs)

### 4. Internationalization (i18n)
- URLs: ES at root (`/`), EN with prefix (`/en/`)
- Use `next-intl` v3.x with `getRequestConfig` and custom middleware
- Message files: `messages/es.json` and `messages/en.json`
- Selector in header: `ES | EN` toggle preserves current route
- Default: Spanish

Configured in `i18n.ts` and `middleware.ts`

### 5. Security (Mandatory)
- Security headers in `next.config.js` (CSP, X-Frame-Options, etc.)
- All environment variables (API keys) prefixed correctly: `NEXT_PUBLIC_*` for client, no prefix for server-only
- `.env.local` never committed â€” `.env.example` always in repo
- No hardcoded secrets anywhere
- IP hashing: Store SHA-256(ip), never plain IPs in Airtable
